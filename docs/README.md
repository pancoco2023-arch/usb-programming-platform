# Plataforma de Práctica de Programación

Este proyecto consiste en el desarrollo de una plataforma web para la práctica de programación, inspirada en plataformas como LeetCode.

## Funcionalidades
- Autenticación de usuarios
- Gestión de problemas
- Ejecución de código
- Validación con casos de prueba

## Tecnologías
- Backend: Flask
- Base de datos: PostgreSQL
- Arquitectura: Cliente-Servidor

## Estructura
- docs/: documentación del proyecto
- src/: código fuente
