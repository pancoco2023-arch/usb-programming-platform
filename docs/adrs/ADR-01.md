# ADR-01 – Framework Backend

## Decisión
Utilizar Flask como framework para el desarrollo del backend.

## Motivo
Flask es un framework ligero basado en Python que permite desarrollar APIs REST de forma sencilla y flexible, adecuado para el alcance de este proyecto educativo.

## Consecuencias
El equipo deberá gestionar manualmente algunos aspectos como la autenticación, que en frameworks más completos ya vienen integrados.
