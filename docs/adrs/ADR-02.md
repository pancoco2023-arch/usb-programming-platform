# ADR-02 – Base de Datos

## Decisión
Utilizar PostgreSQL como base de datos relacional.

## Motivo
PostgreSQL es una base de datos robusta, segura y escalable que permite manejar relaciones complejas entre entidades como las definidas en el modelo entidad-relación.

## Consecuencias
Se requiere configurar un servidor PostgreSQL y gestionar migraciones de esquema a medida que el modelo evolucione.
